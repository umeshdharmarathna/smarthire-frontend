# Smart Hire - Frontend Web Application 🎨

A modern, responsive, glassmorphism-styled web application for hiring trusted local service providers.

---

## 🌟 Tech Stack & Features

* **Technologies**: HTML5, Vanilla CSS3 (Custom Styling & Glassmorphism), Modern ES6+ JavaScript
* **Features**:
  * User Registration & Login (Customer, Provider, Admin)
  * Service Directory & Category Filter
  * Provider Profiles & Instant Booking
  * Customer Booking History & Status Updates
  * Comprehensive Admin Dashboard
  * Interactive AI Help Chat Widget
* **Deployment**: Vercel Static Hosting

## 🛠️ Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/umeshdharmarathna/smarthire-frontend.git
   cd smarthire-frontend
   ```

2. **Run locally**:
   Open `index.html` in any web browser or serve with Live Server / HTTP server:
   ```bash
   npx serve .
   ```
